# docs/ — GitHub Pages static demo

This folder is served as the GitHub Pages site for `monitoring-dashboard`.

## Structure

```
docs/
├── index.html          # Landing page (auto-loaded by GitHub Pages)
└── snapshots/
    ├── china_activity.html
    └── japan_activity.html
```

## Updating snapshots

Snapshots are generated by `scripts/export_snapshot.py`. Run it from the repo root:

```bash
python scripts/export_snapshot.py
```

Then commit and push the updated `docs/snapshots/` files:

```bash
git add docs/snapshots/
git commit -m "refresh snapshot: $(date +%Y-%m-%d)"
git push
```

GitHub Pages will update within ~1 minute.

## Adding a new snapshot

1. Edit `scripts/export_snapshot.py` — add an entry to the `EXPORTS` list.
2. Run the script.
3. Add a link card to `docs/index.html`.
4. Commit and push.

## Enabling GitHub Pages

In the repo settings → Pages:
- **Source**: Deploy from a branch
- **Branch**: `main`
- **Folder**: `/docs`
