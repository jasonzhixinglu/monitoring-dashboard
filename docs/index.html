What this dashboard does

Each country page lets you choose a theme (Activity, Inflation, or PMIs) and a look-back window (2–20 years). The app then:

  • Loads raw Haver time-series data and applies the configured transform (level / YoY% / MoM%)
  • Fits a first-principal-component factor across the indicators using PCA
  • Computes per-indicator surprises (AR(1) one-step-ahead forecast errors)
  • Displays a 2×2 dashboard: factor chart, standardised indicator chart, surprise heatmap, and contributions radial
  • Provides a raw-data table and full-history CSV download

Run locally

git clone https://github.com/jasonzhixinglu/monitoring-dashboard
cd monitoring-dashboard
pip install -r requirements.txt

# Point to your haver-data clone (or clone it side-by-side):
# git clone https://github.com/jasonzhixinglu/haver-data ../haver-data

streamlit run app.py
